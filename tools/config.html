🔑 App Config — API Keys

จัดการ API key ที่ใช้ทั้ง Roadside + Home (เก็บใน Firestore config/app) — แก้ที่นี่ที่เดียว ทุกเครื่องได้ค่าใหม่โดยไม่ต้องแก้โค้ด
เข้าสู่ระบบด้วยบัญชี admin ก่อน จึงจะแก้ key ได้ (อ่านได้โดยไม่ต้อง login)
⚠ ยังไม่ได้ login — ดูได้อย่างเดียว
ต้องเปิด “Places API (New)” + จำกัด HTTP referrer ใน Google Cloud
หมายเหตุ: key ใหม่จะมีผลครั้งถัดไปที่ surveyor เปิดแผนที่ (เกือบทันที เมื่อมีเน็ต) · ออฟไลน์ใช้ค่าที่แคชล่าสุด · ปล่อยช่องว่าง = ใช้ key ที่ฝังในโค้ด (fallback)
