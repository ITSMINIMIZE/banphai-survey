🧰 Banphai Survey — Tools

เครื่องมือสำหรับทดสอบและจัดการระบบ

🏠 กลับเมนูหลัก

🚦 Seed Roadside Data

สร้าง 3 จุดสำรวจ + 1,080 interviews (6 surveyors × 60 ราย) เขียนตรงไปยัง Firestore
⚠ DESTRUCTIVE — ใช้กับ DB ที่ว่าง

🏘️ Seed Home Data

สร้าง 300 households (100 surveyors × 3) + ~750 members + ~2,250 trips
⚠ DESTRUCTIVE — ใช้กับ DB ที่ว่าง

🗺️ Import Zones → zones.js

อัปโหลด shapefile (.zip) / GeoJSON → ตรวจ field บังคับ (N, DISTRICT, D_NAME) → สร้าง zones.js สำหรับ Dashboard OD-matrix
ปลอดภัย — แปลงไฟล์ในเครื่อง ไม่แตะ DB

🔑 App Config — API Keys

จัดการ Google / Longdo API key ของทั้ง 2 แอป (เก็บใน Firestore config/app) — login admin ถึงแก้ได้
ปลอดภัย — login admin เท่านั้น

📍 Pre-seed Places

เติมสถานที่ยอดฮิตในพื้นที่ลง places ก่อนวันสำรวจ → ค้นเจอใน local ทันที ลดการยิง Longdo/Google
ปลอดภัย — เตรียมข้อมูลก่อนสำรวจ

🧹 Cleanup SEED Data

ลบเฉพาะ doc ที่ ID มีคำว่า SEED (ข้อมูลจริงไม่ถูกแตะ) — ใช้ก่อน re-seed · ต้องเปิด delete ใน rules ชั่วคราว
⚠ ลบข้อมูล test ออกจาก Firestore
